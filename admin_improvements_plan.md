# Admin Panel and Signup Enhancements Plan

We are working on the following improvements:
1. **Admin User Profile**: Replace the top-right logout button with a profile logo. Tapping it will open a menu containing **View Profile** and **Log Out**. Build a profile page displaying Name, Phone, Age, Email, and a Change Password option.
2. **Unified Navigation Layout**: Align the Admin Panel navigation with Trackman and Manager layouts on narrow screens by replacing the bottom navigation bar with a side navigation drawer.
3. **Creation Tools**: Add tools to create new Vehicles (inside the Fleet registry tab) and Departments (inside a new Departments manager tab).
4. **Detailed Vehicle Screen**: Redirect vehicle card taps to a separate detail screen showing speed, battery percentage, and a Google-styled location map.
5. **Restrict Deletions**: Remove all deletion options from the Admin Panel interface.
6. **Phone Number on Registration**: Require a phone number during user sign up and persist it in the database.
